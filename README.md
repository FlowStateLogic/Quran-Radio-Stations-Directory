# Quran Radio Stations Directory\n\nA comprehensive directory of 109 Quran radio stations, featuring various reciters and riwayat, provided as a static JSON file for easy integration into applications.\n\n## Data Endpoint\n\nThe station data is available in the following JSON file:\n\n[stations.json](stations.json)\n\n## Structure\n\nThe JSON file contains an array of radio objects, each with the following fields:\n\n- `id`: Unique identifier for the station.\n- `name`: Station name in Arabic.\n- `radio_url`: Direct streaming URL for the radio station.\n- `name_en`: Station name in English.\n- `category`: Category in Arabic (e.g., reciter name).\n- `category_en`: Category in English.\n
